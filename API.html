API(Token, Check)

new API(Token, Check)

API
Parameters:
Name Type Description
Token string Your API Token found at http://api.snowflakedev.cf:8332/dashboard
Check boolean Wheter to check if the API token provided is correct. Returns no error if check is completed Note: if check is enabled, it will increase the request count for the API.
Source:
  • api.js, line 13
Example
const {API} = require("snowflake-api");
const api = new API("YOUR_TOKEN_HERE");
api.chatbot("hello", "Udit", "male").then(console.log);
api.meme().then(console.log)
